# SQL Data Warehouse Project

A modern **SQL Server Data Warehouse** project built using a multi-layer architecture to transform raw business data into analytics-ready datasets.

## Project Overview

This project demonstrates the design and implementation of a scalable **Data Warehouse solution** using **SQL Server**.
The architecture follows the industry-standard **Bronze → Silver → Gold** layered approach for structured data processing and analytics.

## Architecture

### Bronze Layer

* Stores raw data ingested directly from source systems
* Maintains original structure with minimal transformations
* Used as the initial landing zone for data

### Silver Layer

* Performs data cleaning, standardization, and transformation
* Handles missing values, duplicates, and data quality checks
* Creates structured and reliable datasets

### Gold Layer

* Provides business-ready analytical tables
* Optimized for reporting, dashboards, and SQL analytics
* Contains curated KPIs and aggregated business metrics

## Data Sources

The project integrates data from multiple business systems:

* **ERP System** (CSV Files)
* **CRM System** (CSV Files)

## Project Requirements

* Build a scalable SQL-based Data Warehouse
* Implement ETL/ELT data processing pipelines
* Design layered architecture (Bronze, Silver, Gold)
* Clean and transform raw business data
* Create analytical models for reporting and insights
* Enable SQL-based business analytics
